<빅데이터분석_공공데이터포털활용>

1. 주제 : 공공데이터포털 사이트의 데이터를 수집하여 빅데이터 분석 및 시각화하기

2. 내용 : 
	- 인사이트 도출을 위한 특정 주제를 설정하기
	- 공공데이터포털(https://www.data.go.kr/)에서 관련 파일데이터(*.csv) 수집하기
	- 수집한 데이터를 이용하여 데이터 분석하여 인사이트 도출하기(5개 이상)
	- 분석한 데이터를 시각화하기 (2개 이상)

3. 제출형식 : 주피터노트북(*. ipynb), 수집한 파일 데이터(*.csv), 결과 PDF 파일
                  본인명의 폴더 생성하여 제출할 것

4. 폴더명 : 공공포털활용_이름

5. 제출처 : 과제제출폴더(1222_빅데이터_공공데이터활용데이터분석)
